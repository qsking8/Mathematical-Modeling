层次分析法（AHP）步骤：
一、进行一致性检验
      1.计算判断矩阵一致性指标CI
      2.查找对应的平均随机一致性指标RI
      3.计算一致性比例CR = CI / RI.
         如果CR<0.1，则可认为判断矩阵的一致性可以接收；否则需要修正

二、计算权重（或得分）
      三种方法：算术平均法、几何平均法、特征值法（常用）
      算术平均法： 
             1.将判断矩阵按列归一化
             2.将归一化的各列相加（按行求和）
             3.将相加后得到的向量中每个元素除以n即可得到权重向量
      几何平均法：
             1.将矩阵的元素按行相乘得到一个新的列向量
             2.将新向量的每个分量开n次方
             3.对该列向量进行归一化即可得到权重向量
      特征值法（判断矩阵的一致性可以接受）：
             1.求出矩阵的最大特征值及其对应的特征向量
             2.对求出的特征向量进行归一化即可得到权重
